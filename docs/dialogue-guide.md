# Dialog Guide

Through Dialogue, players can interact with NPCs in the game. 

## Quick Start

1. Find dialogue assets in `assets/jsons/roleplay/Dialog`
2. Edit with excel and convert to  csv
3. Add dialog data to `asset_library_dialog.js`
